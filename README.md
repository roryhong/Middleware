# 介紹
將路由的紀錄輸出至終端

## 環境建置
- Visual Studio Code v1.57.1
- Node.js v10.15.0
- Express.js v4.17.1

## 安裝
在終端機輸入指令 Clone 此專案至電腦

    git clone https://github.com/roryhong/Middleware.git

進入專案目錄

    cd Middleware

安裝套件

    npm install

啟動專案

    npm run dev

出現以下訊息即可進入localhost:3000查看網站

    App running on port 3000
